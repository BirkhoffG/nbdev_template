# nbdev template

Use this template to more easily create your [nbdev](https://nbdev.fast.ai/) project.

## Python commands

```
# run training
python -m counterfactual.train
# run evaluation
python -m scripts.evaluate_cfs
```

## Build nbs to module

```
nbdev_build_lib
```

## Update nbs from module
```
nbdev_update_lib
```

## clean notebooks
```
nbdev_clean_nbs
```
